# Calculator App

A simple calculator built with Flutter. Supports addition, subtraction, multiplication, and division with a clean, modern dark UI.

## Features

- Basic arithmetic: `+`, `−`, `×`, `÷`
- Decimal point support
- Backspace (⌫) and Clear (C) buttons
- Division by zero error handling
- Responsive display that scales large numbers

## Getting Started

```bash
flutter pub get
flutter run
```

## Running Tests

```bash
flutter test
```
